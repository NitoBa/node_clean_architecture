# Application to studying Clean Architecture concepts using Node JS and Typescript

## Applications functionalities

- Sign up users
- Sign in users
- Create posts
- List posts from user

## Modules
- Auth
  - [] Sign in user
  - [] Sing up user
  - [] Logout user
  - [] Forgot password

- Profile
  - [] List posts from user
  - [] Create a new post
  - [] Delete a post
  - [] Update a post
